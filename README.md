# `treasure-map`
 treasure-map is a regex script that returns Netflix IDS and episode names. Weigh anchor! Yo-ho-ho!☠️
 
 ## How works?
 - open netflix
 - go to tv show page
 - get all html inside class `.episodeSelector-container`
 - save html to `/html` inside project
 - run project
